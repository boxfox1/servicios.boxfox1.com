SISTEMA BOXFOX1 · NOM-035-STPS
Factores de riesgo psicosocial y entorno organizacional

Diagnóstico, implementación y evidencia documental verificable para auditoría/inspección. Enfoque práctico: instrumentos (según aplique), plan de acción y carpeta de evidencia lista para sostener cumplimiento.

CLAVE

¿Qué es / para qué sirve la NOM-035?

La NOM-035-STPS establece obligaciones para identificar factores de riesgo psicosocial, evaluar el entorno organizacional, aplicar medidas preventivas/correctivas cuando corresponda y conservar evidencia documental verificable.

Qué solicita normalmente la STPS

  • Políticas y acciones de prevención (según aplique).
  • Identificación / evaluación mediante instrumentos.
  • Plan de acción con seguimiento y evidencias.
  • Capacitación / sensibilización (listas, temario, constancias si aplica).
  • Carpeta documental lista para auditoría/inspección.

Objetivo del servicio

  • Implementar un programa de cumplimiento acorde al tamaño y realidad del centro de trabajo.
  • Definir estrategia (diagnóstico + instrumentos según aplique).
  • Ejecutar medición/levantamiento y consolidar resultados.
  • Construir plan de acción (medidas y seguimiento).
  • Entregar evidencia documental organizada para inspección.
PARA QUIÉN ES

¿A quién va dirigido?

A roles responsables del programa, seguimiento y evidencias.

  • RH / Capital Humano
  • Seguridad e Higiene / HSE
  • Gerencia general / Operaciones
  • Supervisores y líderes de área
  • Comité o responsables de cumplimiento
MODALIDAD

Cómo se implementa

Ajustado al alcance, tamaño, turnos y estrategia definida.

  • Implementación en sitio o híbrida.
  • Levantamiento y entrevistas (según alcance).
  • Aplicación de instrumentos / evaluación (según estrategia).
  • Capacitación y plan de acción con seguimiento.
ALCANCE

Alcance según tamaño del centro de trabajo

Estructura típica de implementación para sostener cumplimiento con evidencia.

  • Levantamiento inicial + definición de estrategia.
  • Aplicación de instrumentos / evaluación (según aplique).
  • Reporte ejecutivo + plan de acción.
  • Evidencia documental organizada (carpeta).
ENTREGABLES

Qué entregamos

Documentación lista para auditoría/inspección, con trazabilidad y seguimiento.

  • Diagnóstico y mapa de riesgos psicosociales (según aplique).
  • Plan de intervención / acciones correctivas.
  • Evidencia de capacitación (listas, temario, constancias si aplica).
  • Reporte ejecutivo para dirección.
  • Carpeta de evidencia lista para auditoría/inspección.
BENEFICIOS

Beneficios para la empresa

  • Reducción de riesgos psicosociales y mejora del entorno organizacional (según plan de acción).
  • Cumplimiento normativo y evidencia verificable.
  • Fortalecimiento del entorno organizacional favorable.
  • Mejores prácticas de liderazgo y comunicación interna.
INTEGRACIÓN

Integración dentro de programa anual STPS

La implementación de NOM-035 puede integrarse al Programa Anual de Cumplimiento STPS junto con otras normas aplicables, brigadas internas y capacitación obligatoria.

  • Seguimiento continuo.
  • Evidencia consolidada.
  • Reducción de riesgos normativos.
  • Preparación ante inspección.
